Introduksjon til Node.js

Hva er Node.js?

Node.js er et kraftig open source program som gjør at man kan kjøre JavaScript i terminal uten å gjøre det i en nettleser. Node ble skrevet i 2009 av Ryan Dahl men er nå et samarbeidsprosjekt mellom veldig mange, det kommer fortsatt små oppdateringer månedtlig. Node og er et utrolig mye brukt og viktig verktøy å beherske.

Hvordan laste ned

For å laste ned node gå til denne linken. Node er til både Mac, Windows og Linux og er gratis
Det er anbefalt å laste ned LTS versjonen og ikke current. Current har flere funksjoner men er ofte mindre stabil.

Hva gjør node

Node gjør at alle kan kjøre JavaScript på sin egen PC i terminal, uten å måtte bruke inspect element på en nettleser. Det er mange forskjellige terminaler man kan bruke, både innebygd i skriveprogram og innebygd i PCen. Før i tiden brukte folk konsollen i nettlesere for å bruke JavaScript på egen PC

Hvordan bruke node

Naviger deg frem til mappen med js filen i terminal, dette gjør du ved å bruke "cd" kommandoen (change directory). Du kan bruke kommandoen "ls" underveis for å få opp en liste over alle mappene og filene i mappen du er i. Når du er i mappen med js filen skriver du "node (filnavn)"

Demonstrasjon

Her er hvordan man navigerer seg til et node fil og kjører den på MacOS 

name@user-MacBook-Pro ~ % 
name@user-MacBook-Pro ~ % cd Desktop/Skole/oppgaver
name@user-MacBook-Pro oppgaver %
name@user-MacBook-Pro oppgaver % node oppgave1.js
Hello world!
name@user-MacBook-Pro oppgaver %

Node packet manager

NPM er forkortelse for Node packet manager og er et bibliotek av plugins av forhåndsskrevet kode man kan bruke i prosjektene sine. NPM pakker installeres i terminal og er veldig lett.

Her er to viktige packets alle serverapplikasjoner bruker

  • Express

    Express er en node extension som gjør at det er mulig å sette opp en server og ikke bare ha statisk innhold til nettsiden sin, express er en node pakke du vil se omtrent over alt.

  • Nodemon

    Nodemon er et program som funker akkurat som node, men den er ment for å kjøre serverapplikasjoner istedenfor vanlige scripts. Hver gang en endring skjer i filen nodemon kjører blir serveren startet på nytt.

  • Body-parser

    Bodyparser er en npm packet som gjør at man kan gjøre overgangen fra variabler som sendes gjennom express lettere.

Eksempel på å installere pakker:

For at man skal kunne laste ned pakker må man starte med npm init, som står for initialize. Etter du har initialisert npm vil du bli spurt om masse spørsmål, det går helt fint å hoppe over alle med Enter 

 
name@user-MacBook-Pro oppgaver % npm init

Du kan bruke i istendfor install for å spare tid 

 
name@user-MacBook-Pro oppgaver % npm i express
name@user-MacBook-Pro oppgaver % npm i nodemon
name@user-MacBook-Pro oppgaver % npm i body-parser

Nå som vi har installert pakkene du vil ha kan du bruke "require" funksjonen og bruke dens funksjonalitet

require("express")
const app = express()
